# Joshi Trading | Professional Electrical E-Commerce Solution

A robust, full-stack web application developed to modernize the operations of **Joshi Trading**. This platform bridges the gap between traditional retail and digital commerce, providing a seamless experience for both customers and administrators.

## 🌟 Project Highlights
* **Dual-Interface System:** Integrated a user-centric storefront with a secure, password-protected Admin Panel.
* **Dynamic Data Management:** Real-time synchronization between the MySQL database and the frontend product catalog.
* **Advanced UI/UX:** Implemented smooth-scroll navigation, typewriter effects, and Animate On Scroll (AOS) libraries for a modern feel.
* **Operational Efficiency:** Automated feedback collection and category-based product filtering to enhance user engagement.

## 🛠️ Technical Architecture
* **Core Backend:** PHP (Server-side logic & Session management)
* **Database:** MySQL (Relational data modeling for products, orders, and reviews)
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Deployment:** Cloud-hosted via InfinityFree with SSL configuration.

## 📈 Key Modules
1.  **Product Discovery:** Optimized search and category-wise browsing (Fans, Wires, Lighting, Tools).
2.  **Admin Suite:** Full CRUD (Create, Read, Update, Delete) capabilities for inventory management.
3.  **Engagement Hub:** Interactive feedback slider and rating system.
4.  **Accessibility:** Fully responsive design tested across multiple device breakpoints.

## ⚙️ Installation & Usage
1.  Download the source code.
2.  Import `joshi_trading_pro.sql` to your local/live MySQL server.
3.  Update `includes/db.php` with your environment credentials.
4.  Launch via XAMPP or any PHP-supported web server.
